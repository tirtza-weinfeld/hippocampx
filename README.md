# HippocampX

Interactive educational web app with rich content modules.

## Features

- **Problem Sets** — Curated challenges with solutions
- **Dictionary** — Lexical database with semantic search
- **Learning Modules** — MDX content with KaTeX math rendering
- **Notes** — Concept reference library
- **Schema Explorer** — Visual ER diagrams

## Tech

Next.js 16 · React 19 · TypeScript · Tailwind 4 · MDX · Drizzle ORM · PostgreSQL
