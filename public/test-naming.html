Naming Convention Comparison

Same visual styles, different class names:

Current (custom)

class="control primary heavy"

Radix UI style

class="control primary solid"

Material 3 style

class="control primary filled"

Usage in your code

// Current
<button class="control critical heavy">
<div class="surface warning light">

// With Radix naming
<button class="control critical solid">
<div class="surface warning soft">

// With Material naming
<button class="control critical filled">
<div class="surface warning tonal">

All three produce identical visuals. Pick whichever vocabulary feels right to you.
