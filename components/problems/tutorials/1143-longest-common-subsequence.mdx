## [!(ProblemDefinition)] Definition

Given two strings , return the length of their longest common subsequence

## Top Down Suffix Solution

### [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/1143-longest-common-subsequence/top-down-suffix.py"
def longestCommonSubsequence(text1: str, text2: str) -> int:
    memo = {}

    def dp(i, j):
        if i >= len(text1) or j >= len(text2):
            return 0
        if (k := (i, j)) not in memo:
            if text1[i] == text2[j]:
                memo[k] = 1 + dp(i + 1, j + 1)
            else:
                memo[k] = max(dp(i + 1, j), dp(i, j + 1))
        return memo[k]

    return dp(0, 0)
```

## Bottom Up Suffix Solution

### [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/1143-longest-common-subsequence/bottom-up-suffix.py"
def longestCommonSubsequence( text1: str, text2: str) -> int:
    n1, n2 = len(text1), len(text2)
    dp = [[0] * (n2+1) for _ in range(n1+1)]

    for i in reversed(range(n1)):
        for j in reversed(range(n2)):
            if text1[i] == text2[j]:
                dp[i][j] = 1 + dp[i + 1][j + 1]
            else:
                dp[i][j] = max(dp[i + 1][j], dp[i][j + 1])
    return dp[0][0]
```

## Bottom Up Prefix 1d Solution

### [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/1143-longest-common-subsequence/bottom-up-prefix-1d.py"
def longestCommonSubsequence(a: str, b: str) -> int:
    if len(a) < len(b):
        a, b = b, a
    dp = [0] * (len(b) + 1)
    for ca in a:
        diag = 0
        for j, cb in enumerate(b, 1):
            diag, dp[j] = dp[j], (diag + 1 if ca == cb else max(dp[j], dp[j - 1]))
    return dp[-1]
```
