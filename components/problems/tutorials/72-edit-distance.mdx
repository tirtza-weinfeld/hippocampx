## [!(ProblemDefinition)] Definition

Given two strings a and b, return the minimum number of operations required to convert a to b(insert,delete,replace)

## [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/72-edit-distance/solution.py"
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:

        n1, n2, memo = len(word1), len(word2), {}

        def dp(i, j):
            if i >= n1: return n2 - j
            if j >= n2: return n1 - i
            if (k := (i, j)) not in memo:
                memo[k] = ( 
                    dp(i + 1, j + 1) if word1[i] == word2[j] 
                    else 1 + min(dp(i + 1, j), dp(i, j + 1), dp(i + 1, j + 1))
                )
            return memo[k]

        return dp(0, 0)
```
