## [!(ProblemDefinition)] Definition

Given an array of integers and a target k, determine if there exists a subarray with sum divisible by k.

## [!(ProblemArguments)] Key Arguments

- `segments: list[int]`: List of integers representing trip segments.
- `k: int`: The divisor.

## [!(ProblemReturns)] Returns

`bool`: True if such a subarray exists, False otherwise.

## [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/523-continuous-subarray-sum/solution.py"
def checkSubarraySumIsMultipleOfK(segments: list[int], k: int) -> bool:

    remainder_idx = {0: -1}
    prefix_remainder = 0

    for i, segment in enumerate(segments):
        prefix_remainder = (prefix_remainder + segment) % k
        if prefix_remainder in remainder_idx:
            if i - remainder_idx[prefix_remainder] > 1:
                return True
        else:
            remainder_idx[prefix_remainder] = i
    return False
```
