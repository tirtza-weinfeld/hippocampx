[35. Search Insert Position](https://leetcode.com/problems/search-insert-position)

## [!(ProblemDefinition)] Definition

Given a sorted array of distinct integers and a target value, find the index of the target. If absent, return the index where it would be inserted to maintain order.

## [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/35-search-insert-position/solution.py"
from bisect import bisect_left

def search_insert_position(nums: list[int], target: int) -> int:
    return bisect_left(nums, target)
```
