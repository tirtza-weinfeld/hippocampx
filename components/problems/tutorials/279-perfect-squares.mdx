## [!(ProblemDefinition)] Definition

Given an integer `n`, return the minimum number of perfect squares whose sum equals `n`.
    A perfect square is an integer of the form $k^2$ for some integer `k` (e.g., 1,4,9,16).

## [!(ProblemTimeComplexity)] Time Complexity

- $O(n\sqrt{n})$:
    - $\sum_{i=1}^{n} \sqrt{i} = O(n\sqrt{n})$

## [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/279-perfect-squares/solution.py"
from math import isqrt

def numSquares(n: int) -> int:
    dp = [0] + [n] * n
    for i in range(1, n + 1):
        dp[i] = min(dp[i - j*j] + 1 for j in range(1, isqrt(i) + 1))
    return dp[n]
```
