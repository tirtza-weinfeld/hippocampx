## [!(ProblemDefinition)] Definition

Given a string s of '(' , ')' and lowercase English characters.emove the minimum number of parentheses ( '(' or ')', in any positions ) so that the resulting parentheses string is valid and return any valid string.

## [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/1249-minimum-remove-to-make-valid-parentheses/solution.py"
def minRemoveToMakeValid(s: str) -> str:
    stack, remove = [], set()
    for i, c in enumerate(s):
        if c == "(":
            stack.append(i)
        elif c == ")":
            if stack:
                stack.pop()
            else:
                remove.add(i)
    remove.update(stack)
    return "".join(c for i, c in enumerate(s) if i not in remove)
```
