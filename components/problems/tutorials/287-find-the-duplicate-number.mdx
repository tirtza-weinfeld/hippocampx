## [!(ProblemDefinition)] Definition

Given an array of integers nums containing n + 1 integers where each integer is in the range [1, n] inclusive.There is only one repeated number in nums, return this repeated number.You must solve the problem without modifying the array nums and using only constant extra space.

## [!(ProblemIntuition)] Intuition

- Finds the duplicate in $O(N)$ time and $O(1)$ space using
    - Floyd's Cycle-Finding Algorithm.
    - [The Hare and Tortoise Algorithm](/notes/hare-and-tortoise-algorithm)

## [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/287-find-the-duplicate-number/solution.py"
def findDuplicate(nums: list[int]) -> int:
    slow = fast = 0
    while (slow := nums[slow]) != (fast := nums[nums[fast]]):
        pass
    slow = 0
    while (slow := nums[slow]) != (fast := nums[fast]):
        pass
    return slow
```
