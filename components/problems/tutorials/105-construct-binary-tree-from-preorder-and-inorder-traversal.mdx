## [!(ProblemDefinition)] Definition

Given two integer arrays preorder and inorder where preorder is the preorder traversal of a binary tree
    and inorder is the inorder traversal of the same tree, construct and return the binary tree.
    preorder and inorder consist of unique values.

## [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/105-construct-binary-tree-from-preorder-and-inorder-traversal/solution.py"
from backend.algorithms.core.Tree.binary_tree_node import TreeNode

class Solution:
    def buildTree(self, preorder: list[int], inorder: list[int]) -> TreeNode | None:
        it = iter(preorder)
        idx = {v: i for i, v in enumerate(inorder)}

        def build(l: int, r: int) -> TreeNode | None:
            if l > r:
                return None

            v = next(it)
            m = idx[v]

            return TreeNode(
                v,
                build(l, m - 1),
                build(m + 1, r),
            )

        return build(0, len(inorder) - 1)
```
