## [!(ProblemDefinition)] Definition

Given a list of daily stock prices `prices`; choose one day to buy and a later day to sell for maximum profit. Return the max profit, or `0` if no profit is possible.

## [!(ProblemTimeComplexity)] Time Complexity

- $O(n)$
    - where n is the length of the prices array. We iterate through the array once.

## [!(ProblemReturns)] Returns

`int`: Maximum profit from buying and selling stock

## [!(ProblemCodeSnippet)] Code Snippet

```python meta="source=problems/121-best-time-to-buy-and-sell-stock/solution.py"
def best_time_to_buy_and_sell_stock(prices: list[int]) -> int:
    cost, profit = float("inf"), 0
    for price in prices:
        cost = min(cost, price)
        profit = max(profit, price - cost)
    return profit
```
