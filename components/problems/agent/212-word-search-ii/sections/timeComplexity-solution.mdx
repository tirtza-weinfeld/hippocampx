- $O(m n \cdot 4^L)$
    - **Symbols:** m×n board, W=∑(w∈dict)|w|, L=max|w|.
    - **Build trie:** $O(W)$.
    - **DFS (worst case):** first step ≤ 4 choices, then ≤ 3 per step ⇒ $O(m n \cdot 4 * 3^(L-1)$). Coarse bound: $O(m n \cdot 4^L)$.
    - **Space:** trie $O(W)$; recursion/visited path $O(L)$.
