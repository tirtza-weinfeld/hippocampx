# Gradient Guide

A **gradient** is an image created using CSS. Gradients allow smooth transitions between colors, defined by angles, color stops, and layers.

## Gradient Types

1. **Linear Gradient**: Smooth transition along a straight line.
2. **Radial Gradient**: Transition radiates outward from a central point.
3. **Conic Gradient**: Transition is circular around a central point.

## Linear Gradient

### Example
```css
background: linear-gradient(black, white);
```
<GradientExample type="linear" colors={['black', 'white']} angle="to bottom" />

### With Angle
```css
background: linear-gradient(90deg, black, white);
```
<GradientExample type="linear" colors={['black', 'white']} angle="90deg" />

## Color Stops

### Example
```css
background: linear-gradient(0deg, red, blue);
```
<GradientExample type="linear" colors={['red', 'blue']} angle="0deg" />

## Multiple Gradients

### Example
```css
background: linear-gradient(black, white), linear-gradient(red, blue);
```
