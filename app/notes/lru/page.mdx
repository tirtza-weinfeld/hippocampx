# LRU Cache

## Table of Contents


## Intuition
Use an OrderedDict (hash map + doubly-linked list):
* on every get or put you move the key to the end (most-recent); 
* when over capacity you popitem( <CodeStep step={1}>last=False</CodeStep> ) to drop the front (least-recent).

## Time Complexity
Both get and put run in `O(1)` average time (dict lookup/update + constant-time list splicing and eviction).



## Code Snippet
```python meta="{5}/last//move_to_end/ lru.py"
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.cache: OrderedDict[int, int] = OrderedDict()
        self.capacity = capacity

    def get(self, key: int) -> int:
        if (val := self.cache.get(key)) is None:
            return -1
        self.cache.move_to_end(key)
        return val

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)      
```
