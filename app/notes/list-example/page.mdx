This page demonstrates the various list components available in our MDX setup.

## Unordered Lists

Basic unordered list:

- First item
- Second item
- Third item with **bold text**
- Fourth item with *italic text*

## Nested Lists

Lists can be nested:

- Main category
  - Subcategory A
    - Detail 1
    - Detail 2
  - Subcategory B
    - Detail 3
    - Detail 4
- Another main category
  - Subcategory C
  - Subcategory D

## Mixed Content Lists

Lists can contain various content types:

- **Bold text** and regular text
- `Inline code` and more text
- [Links](https://example.com) within list items
- Multiple paragraphs:
  
  This is a second paragraph within a list item.
  
  And a third paragraph.

## Code Examples

Here's how you can use lists in your MDX files:

### Unordered List
```markdown
- Item 1
- Item 2
- Item 3
```

### Ordered List
```markdown
1. First step
2. Second step
3. Third step
```

### Nested List
```markdown
- Main item
  - Sub item
    - Sub-sub item
  - Another sub item
```

## Features

The list components automatically provide:
- Smooth animations with Framer Motion
- Dark mode support
- Responsive design
- Accessible markup
- Custom styling with Tailwind CSS
- Support for nested content
- Proper spacing and typography
