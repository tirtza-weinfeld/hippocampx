## Number Systems

* **Natural ($\mathbb{N}$)**: $\{1, 2, 3, \ldots\}$ → counting samples, epochs.
* **Integers ($\mathbb{Z}$)**: $\{... -2, -1, 0, 1, \ldots\}$ → signed updates.
* **Rationals ($\mathbb{Q}$)**: a/b → probabilities, ratios.
* **Reals ($\mathbb{R}$)**: all rationals + irrationals → gradients, losses.
* **Complex ($\mathbb{C}$)**: a + bi → Fourier, signal models.

##  Operations & Properties

* **Closure**: $\mathbb{Z}$ closed under $+, -, \times$.
* **Commutative**: a + b = b + a.
* **Associative**: (a + b)+c = a+(b + c).
* **Distributive**: a(b + c) = ab + ac.
* **Example**: $(2 \times 3) + (2 \times 4) = 14 = 2 \times (3 + 4)$.

## **Part 3 – Fractions, Exponents, Logs**

* **Simplify**: 6/9 = 2/3.
* **Exponent rules**: aᵐ·aⁿ = aᵐ⁺ⁿ, (aᵐ)ⁿ = aᵐⁿ.
* **Log laws**: log(ab)=log a+log b.
* **AI use**: log-likelihood, softmax stability (`log sum exp`).

## **Part 4 – Sequences & Series**

* **Arithmetic**: aₙ = a₁ + (n–1)d → linear growth.
* **Geometric**: aₙ = a₁ rⁿ⁻¹ → exponential learning curves.
* **Sum**: S = a₁(1 – rⁿ)/(1 – r).
* **Use**: convergence, learning-rate decay.

## **Part 5 – Functions & Relations**

* **Function**: maps x→y (unique output).
* **Linear**: y = ax + b → weights & bias.
* **Exponential**: y = eˣ → activations.
* **Piecewise/ReLU**: y = max(0, x) → neural nets.
