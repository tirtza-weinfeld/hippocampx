## Definition

**$\mathbb{R}$** (the symbol **R** but written as **$\mathbb{R}$**) means **the set of all real numbers**.

A **real number** is any number that can be placed on an infinite straight number line.

**Core idea:**
A real number is any quantity that has a definite position on the continuous number line — no gaps.




**Includes:**

* **Integers:** …, –2, –1, 0, 1, 2
* **Fractions:** 1/2, –7/3
* **Decimals:** 3.14, –0.001
* **Irrationals:** √2, π, e

**Excludes:**

* Imaginary numbers like $ i = \sqrt{-1} $
